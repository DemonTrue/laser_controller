### STM32 Laser Controller

Система управления лазерным модулем с поддержкой различных режимов генерации импульсов, контролем температуры (Пельтье/вентилятор) и интерфейсом на базе LCD дисплея HD44780

### Особенности:
- Генерация коротких (мкс/мс), длительных (с) и непрерывных комплементарных импульсов с программируемым deadtime для управления силовыми ключами
- Управление системой охлаждения (вентилятор и элемент Пельтье) через ШИМ
- Настройка параметров эксперимента через энкодер и кнопки
- Отображение текущего состояния, времени и настроен на LCD дисплее 1602
- Запуска эксперимента кнопкой или тумблером
- Аварийное прерывание эксперимента с сохранением безопасности
